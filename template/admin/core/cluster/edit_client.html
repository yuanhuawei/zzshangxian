{$P8LANG['cluster_client']['name']} 请添加分站名称，名称可任意填写
{$P8LANG['cluster_client']['url']} 请填写分站的网站，格式：http://www.xxx.com 或 http://xxx.com

{$P8LANG['cluster_client']['charset']} 根据分站自身站点编码，一般为gbk 或 utf8
