IP禁止设置
是否启用IP禁止: 开启 不开启  
禁止IP: 你可以设置封掉设置过的IP访问网站前台和后台,每个IP换一行,书写格式如下：
192.168.0.2
192.168.1.5
...
192.168.0 格式表示限制所有192.168.0.0--192.168.0.254段的IP
注意:这里会同时禁止访问前台与后台,请不要封自己的IP,否则很麻烦
禁止IP段: -- 例如：192.168.0.2-192.168.0.111
