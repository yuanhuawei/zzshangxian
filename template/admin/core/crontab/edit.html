1、任务脚本栏只需填写计划任务文件名即可，你先将自己所编的计划任务程序文件用FTP传至系统。
2、里面的计划任务参数根据需要填写，不一定要全部填写。 任务执行时间规律
小时与分钟具有最高优先权。这两个设置一定有效。如只设这两个值，则为每隔X小时/X分钟执行。
优先先 每隔几天 > 星期几 > 每月几号
如果设置了上面其中一个值。那么小时及分钟则为执行当天的小时。
例:
天: 0, 星期: 0, 几号: 0, 小时: 3 (72) -> 每隔3 (72)小时执行
天: 0, 星期: 0, 几号: 0, 小时: 0, 分钟: 5 -> 每隔5分钟执行
天: 60, 星期: 0, 几号: 0, 小时: 12, 分钟: 0 -> 每隔60天的12点执行
天: 0, 星期: 5, 几号: 0, 小时: 12, 分钟: 0 -> 每星期5的12点执行
天: 0, 星期: 0, 几号: 15, 小时: 12, 分钟: 0 -> 每月15号的12点执行
