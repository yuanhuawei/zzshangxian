全局ip白名单设置
允许IP: 注意:访问的客户端必须是固定IP或者在固定的IP段内
否则很有可能导致客户端无法正常访问,此时只能通过修改后台文件强制进入后台进行修改恢复
书写格式如下：
192.168.0.2
192.168.0.5
...
允许IP段: -- 例如：192.168.0.2-192.168.0.111
允许IP段内例外IP: 附注:在允许IP段内，可以限制排除下列IP地址
书写格式如下：
192.168.0.2
192.168.0.5
...

允许登录后台IP段: -- 例如：192.168.0.2-192.168.0.111
允许登录后台IP: 写错很有可能导致进入后台
书写格式如下：
192.168.0.2
192.168.0.5
...
